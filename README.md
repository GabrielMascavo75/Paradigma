# Jogo de Adivinhação
##Introdução
O objetivo desse projeto é demonstrar conceitos de programação orientado a objetos (poo) utilizando linguagem python. O jogador deve advinhar o número gerado aleatoriamente, com dicas fornecidas pelo sistema.

##Conceito de POO utilizados
Classe abtrata(Jogo)  
Herança (JogoAdvinhacao herda de Jogo)  
Encapsulamento (__tentativas e __limite)  
Objetos (Jogador)
